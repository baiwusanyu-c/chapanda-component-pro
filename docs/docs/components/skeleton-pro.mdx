## API

### Props

| Name           | Type                                             | Default    | Description                                                                                   |
| -------------- | ------------------------------------------------ | ---------- | --------------------------------------------------------------------------------------------- |
| type           | `"list" \| "description" \| "matrix" \| "detail" \| "table" \| "query"`| `-`        | 标记骨架屏渲染类型 |
| active         | `boolean`                                         | `true`        | 是否显示动态          |
| renderNum      | `number`                                         | `-`        |  渲染数量， 仅 `type='list'`或 `type='description'`或 `type='matrix'`生效  |
| rowProps       | `RowProps`                                         | `-`        | `row` 组件 `props`， 仅 `type='matrix'` 生效 |
| colProps       | `ColProps`                          | `fixed`    | `col` 组件 `props`， 仅 `type='matrix'` 生效                                                                    |
| renderTableRow   | `number` | `-` | 渲染 `table` 的行数、仅 `type='detail'`或 `type='table'`生效                                                     |
| renderTableCol    | `number`                                         | `-`       | 渲染 `table` 的列数、仅 `type='detail'`或 `type='table'`生效                                                                 |
| renderTableQuery  | `boolean`                         | `true`       | 渲染 `table` 的 `query` 区域、仅 `type='detail'`或 `type='table'`生效                                                                     |
| renderDetailTable   | `boolean`                         | `true`       | 渲染 `detail` 的 `table` 区域, 仅 `type='detail'` 生效                                                                         |
| skeletonProps        | `SkeletonProps`                         | `-`       | 作用在每个原子 `Skeleton` 组件上的 `props`                                                                         |
