## API

### Props

| Name           | Type                                             | Default    | Description                                                                                   |
| -------------- | ---------------------------- | ---------- | ----------------- |
| isATag         | `boolean`| `-`               | 是否渲染成 `a` 标签 |
| attrs          | `Record<string, any>`        | `-`        | 继承到渲染元素上的属性    |
| titleIcon      | `React.ReactNode`            | `-`        | 标题图标  |
| title          | `string \| React.ReactNode`  | `-`        | 标题  |
| extra          | `string \| React.ReactNode`  | `-`        | 标题的附加内容 |
| showExtra      | `boolean`                    | `true`    | 是否显示标题附加内容                                                                    |
| disabled       | `boolean`                    | `false` | 是否禁用 |
| description    | `string \| React.ReactNode`  | `-`       | 描述内容              |
