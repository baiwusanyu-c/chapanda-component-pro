# 快速开始

`chapanda component pro` 是基于 `antd` 的业务组件二次封装

## 安装

<PackageManagerTabs command="@chapanda/component-pro " />

## 开始使用

在应用的入口文件 `main.tsx` 引入必要的组件样式

```typescript jsx
import "@chapanda/component-pro/react/styles/table-pro/index.css";
import "@chapanda/component-pro/react/styles/layout-title/index.css";
import "@chapanda/component-pro/react/styles/nav-card/index.css";
import "@chapanda/component-pro/react/styles/skeleton-pro/index.css";
```

在应用的根组件文件 `app.tsx` 引入 `ChapandaProvider` 组件，它将为 `component pro` 的其他组件提供必要的上下文事件，以确保功能的正常执行


```typescript jsx
import React from "react";
import { ConfigProvider } from "antd";
import zhCN from "antd/locale/zh_CN";
import { ChapandaProvider } from "@chapanda/component-pro";
import RouterView from "./router";

export default (() => {
	return (
		<ConfigProvider
				componentSize="middle"
				locale={zhCN}>
				<ChapandaProvider>{RouterView()}</ChapandaProvider>
		</ConfigProvider>
	);
});

```
