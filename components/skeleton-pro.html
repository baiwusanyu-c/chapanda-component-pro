#SkeletonPro

SkeletonPro 是chapanda component pro 基于 antd 的业务组件二次封装业务组件。

#样例

#基础使用

将 type 属性设置为 list, 可以渲染简单的骨架列表，renderNum 可以设置渲染行数。

#开启动画

将 active 属性设置为 true, 开启动画。

#描述列表

将 type 属性设置为 description, 可以渲染为描述列表骨架屏。

#卡片矩阵

将 type 属性设置为 matrix, 可以渲染为描述列表骨架屏。

#查询骨架

将 type 属性设置为 query, 可以渲染为查询骨架屏。

#表格查询

将 type 属性设置为 table, 可以渲染为表格骨架屏。

#详情页骨架

将 type 属性设置为 detail, 可以渲染为详情页骨架屏。
